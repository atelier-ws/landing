# Atelier

Canonical LLM summary: https://atelier.ws/llms.txt

Atelier is an open-source context runtime for coding agents. It provides an MCP server and SDK middleware for code intelligence, context reuse, model routing, cost tracking, failure rescue, loop detection, and auditable traces.

Important links:

- Website: https://atelier.ws/
- Markdown mirror: https://atelier.ws/index.md
- Docs: https://docs.atelier.ws/installation
- Source: https://github.com/atelier-ws/atelier
- Contact: contact@atelier.ws

Install:

```bash
curl -fsSL https://install.atelier.ws | bash
```
